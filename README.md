🚀 Installation & Setup
1-Install the coppyApp folder

2-extract the downloaded folder

3-click on the copp.exe file to dowload the app

📖 How to Use
1-Ensure the application is running. You will see the Coppy icon in your system tray (bottom right of your screen).

2-Click the system tray icon to activate the selection overlay.

3-Click and drag your mouse to draw a rectangle over the text you want to extract.

4-Release the mouse. The text is now on your clipboard and ready to be pasted!
